# Organization Profile Repo (.github)

This repo powers your **organization profile page** on GitHub via `profile/README.md`.

## Files here
- `sitemap.xml` — template (note: search engines don't read sitemaps from regular GitHub repos).
- `google25ab32e9faa1d89e.html` — example Google verification file (replace the code with your own; works on **GitHub Pages** sites you control, not `github.com`).
- `profile/README.md` — this is what shows on your org profile page.
- `profile/sitemap.xml` — duplicate of the sitemap template.
- `profile/google25ab32e9faa1d89e.html` — duplicate verification file.

## Replace placeholders
- Replace all `YOUR-ORG` with your **actual GitHub organization name**.
- If you have your own verification code, rename `google25ab32e9faa1d89e.html` to your code and update the contents.

## Important SEO note
To actually use `sitemap.xml` or Google verification, put them in a **GitHub Pages** site (like `YOUR-ORG.github.io`) or a custom domain — not just this `.github` repo.
